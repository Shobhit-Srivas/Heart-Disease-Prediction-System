# 🫀 Heart Disease Prediction System

## 📌 Project Overview

Heart Disease Prediction System is a Machine Learning-based healthcare application developed using Python, Scikit-Learn, and Streamlit. The application predicts whether a person is at high or low risk of heart disease based on various medical parameters.

The project utilizes a trained K-Nearest Neighbors (KNN) model and provides predictions through an interactive Streamlit web application.

## 📊 Input Parameters

The model predicts heart disease risk using:

* Age
* Gender
* Chest Pain Type
* Resting Blood Pressure
* Cholesterol Level
* Fasting Blood Sugar
* Resting ECG Results
* Maximum Heart Rate
* Exercise-Induced Angina
* Oldpeak (ST Depression)
* ST Slope

---

## 🧠 Machine Learning Workflow

### Step 1: Data Collection

Medical records containing heart disease information were collected and analyzed.

### Step 2: Data Preprocessing

* Data Cleaning
* Feature Engineering
* Encoding Categorical Variables
* Feature Scaling

### Step 3: Model Training

A K-Nearest Neighbors (KNN) model was trained using the processed dataset.

### Step 4: Model Evaluation

The model was evaluated using classification metrics to ensure reliable predictions.

### Step 5: Deployment

The trained model was integrated into a Streamlit web application.

## ▶️ Installation

### Clone Repository

```bash

```

### Move Into Project Folder

```bash

```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Streamlit App

```bash
streamlit run app.py
```
